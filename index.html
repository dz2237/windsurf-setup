Windsurf 环境配置指南

配置步骤

1. 安装 Python 3.11

  1. 访问 Python 3.11.0 下载页面
  2. 根据您的操作系统选择合适的安装包：
    • Windows: 下载 "Windows installer (64-bit)"
    • macOS: 下载 "macOS 64-bit universal2 installer"
  3. 运行下载的安装程序
  4. 重要：安装时勾选 "Add Python 3.11 to PATH"
  5. 完成安装后，打开命令行（Windows按Win+R输入cmd，macOS打开终端）
  6. 输入 python --version 确认安装成功，应显示 Python 3.11.x

2. 安装必要的 Python 库

  1. 在命令行中输入以下命令安装 numpy： 
    pip install numpy
  2. 安装 pandas： 
    pip install pandas
  3. 验证安装： 
    python -c "import numpy; import pandas; print('numpy version:', numpy.__version__); print('pandas version:', pandas.__version__)"

3. 安装 Windsurf

  1. 在命令行中输入： 
    pip install windsurf
  2. 等待安装完成
  3. 验证安装： 
    python -c "import windsurf; print('windsurf installed successfully')"
